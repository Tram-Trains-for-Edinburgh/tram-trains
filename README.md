# Tram Trains for Edinburgh

Campaign website and tools for [Tram Trains for Edinburgh](https://www.ttfe.org.uk).

- **[pf-site/](pf-site/)** — The website (Eleventy + Decap CMS, hosted on Cloudflare Pages)
- **[oauth-worker/](oauth-worker/)** — Cloudflare Worker for CMS authentication
- **[map/](map/)** — Route map assets

See [pf-site/README.md](pf-site/README.md) for full setup and development instructions.
